II. Review Questions: 

    1. What is Semantic HTML? 
		Use of markups to reinforce meaning.

    2. What is HTML used for? 
		HTML is markup language and it is used for creating web pages.


    3. What is an attribute and where do we put it? 
		Attributes are special words inside an element tag to control the elements behavior.
		

    4. What is the h1 tag used for? How many times should I use it on a page?
		h1 tag is usually used for the title, and should only be used once per page for SEO reasons.
		
    5. Name two tags that have required attributes
		<a> tag defines a hyperlink, which is used to link from one page to another. 
		<img> Defines an image in an HTML page. Two required attributes are src and alt.
		
    6. What do we put in the head of our HTML document? 
		<head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag. 
		 HTML metadata is data about the HTML document. Metadata typically definea the document title, character set, styles, scripts, and other meta information.

    7. What is an id? 
		The id attribute is most used to point to a style in a style sheet, and by JavaScript (via the HTML DOM) to manipulate the element with the specific id.

    8. What elements can I add an id to? 
		 The id attribute can be used on any HTML element.

    9. How many times can I use the same id on a page? 
		 id can be used once per page.

    10. What is a class? 
		Class in HTML is used to refer CSS (Cascading style sheet), to which we apply some style or property in CSS file. 
		You can define a class that comprises of either a single or more than one (multiple) HTML elements that should have a particular style for property in common.

    11. What elements can I add a class to? 
		The class attribute can be used on any HTML element.

    12. How many times can I use the same class on a page?
			You can use a class attribute on a page as many times as you need to.
 
    13. How do I get my link to open in a new tab?
		If you set the target attribute in the <a> tage to "_blank", the link will open in a new browser window or a new tab.

    14. What is the alt attribute used for? 
		alt attribute specifies an alternate text for an image.

    15. How do I reference an id?
		html example:
                              <h1 id="example-id">I have an ID</h1>

		css example:

				#example-id {  
  					color: gold;  
				}  	
		
    16. What is the difference between a section and a div

		The section tag defines sections in a document like chapters, headers, footers whereas div tag defines a division in an HTML document.
	        <section> gathers a specific and strong semantic purpose, where, div has only a generic, non-specific meaning and no semantic purpose.


    17. What is CSS used for? 
		
		CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts.
	        It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers. 
		CSS is independent of HTML and can be used with any XML-based markup language.
	
    18. How to we select an element? Example - every h2 on the page

		You select an element by referencing it through CSS. for example: h2{color:red;}   - This example references every h2 and changes the color of the font to red.
		
    19. What is the difference between a class and an id? - Give me an example of when I might use each one

    20. How do we select classes in CSS?
			
		In the CSS, a class selector is a name preceded by a full stop (“.”) and an ID selector is a name preceded by a hash character (“#”). 
				
    21. How do we select a p element with a single class of “human””?
		.human p{....}
		
    22. What is a parent child selector? When would this be useful? 
		CSS inheritance is similar to human inheritance in that you can pass traits down to ancestor elements. 
		In CSS, every ancestor of the original parent gains access to all the CSS properties through inheritance. 
		It is usefull because of specificity and inheritance. Specificity is the concept we use to describe the ability to style many elements or just one element using CSS selectors. 		
		
    23. How do you select all links within a div with the class of sidebar?
				.sidebar{...}

    24. What is a pseudo selector?
		A pseudo-class is a selector that assists in the selection of something that cannot be expressed by a simple selector.
		Pseudo-classes are used when you may not have a set amount of elements, but you still want to style them individually.	
			
			Example:

			h1:last-child {  
  				color: orange;  
					}

				You can't put block elements inside inline elements. Difference is also formatting. By default, inline elements do not force a new line to begin in the document flow.

				An absolutely positioned element is an element whose computed position value is absolute or fixed. The top , right , bottom , and left properties specify offsets from the edges of the element's containing block.
